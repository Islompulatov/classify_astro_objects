# MLOps - Celestial Body Classification

This repository contains the code from
[A guide to MLOps](https://mlops.swiss-ai-center.ch/).

## Notes
This repository hosts an end-to-end MLOps pipeline designed to automate the data collection, preprocessing, training, evaluation, and containerized deployment of a machine learning model that classifies astronomical objects (stars, galaxies, and quasars) based on spectral and photometric data.

The project demonstrates production-grade machine learning engineering by treating the model not just as a static script, but as an automated, observable, and reproducible software system.
